# Mammoth – Automate data transformations with data versioning and lineage

Mammoth is a fork of Pachyderm, focused on being a fully open-source, enterprise-free data versioning and lineage platform. It enables data engineering teams to automate complex pipelines with sophisticated data transformations across any type of data, with data versioning and data lineage tracking.

## Features
- Data-driven pipelines automatically trigger based on detecting data changes.
- Immutable data lineage with data versioning of any data type. 
- Autoscaling and parallel processing built on Kubernetes for resource orchestration.
- Uses standard object stores for data storage with automatic deduplication.  
- Runs across all major cloud providers and on-premises installations.
